# diploma
My bachelor diploma

## Очистка докера

```
docker system prune	-f
docker volume prune -f
```

## Установка проекта

```
npm install
```

## Запуск проекта в среде разработки

```
npm run serve
```

## Запуск форматтера

```
npm run prettier
```

## Запуск БД

```
docker-compose up --build
```
